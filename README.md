# Password Security Analyzer Using Python

## Overview

Password Security Analyzer is a Python-based cybersecurity project that evaluates the strength of a user's password and provides security recommendations. The system analyzes password complexity, calculates entropy, detects weak patterns, and classifies passwords into different security levels.

## Features

* Password Strength Analysis
* Entropy Calculation
* Detection of Common Passwords
* Detection of Repeated Characters
* Detection of Sequential Patterns
* Security Score Generation
* Password Classification
* Security Recommendations

## Technologies Used

* Python
* Regular Expressions (Regex)
* Math Library

## Project Workflow

1. User enters a password.
2. System checks password length.
3. System verifies uppercase and lowercase letters.
4. System checks numbers and special characters.
5. Common passwords are detected.
6. Sequential and repeated patterns are identified.
7. Password entropy is calculated.
8. Security score is generated.
9. Password strength is classified.
10. Security recommendations are displayed.

## Password Strength Levels

| Score Range | Strength    |
| ----------- | ----------- |
| 0 - 39      | Weak        |
| 40 - 69     | Medium      |
| 70 - 89     | Strong      |
| 90 - 100    | Very Strong |
